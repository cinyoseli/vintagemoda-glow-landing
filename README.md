# Landing page – vintagemoda.glow

Landing page para emprendimiento de moda vintage, diseñada para mostrar productos y facilitar el contacto directo por WhatsApp.

## Tecnologías utilizadas
- HTML
- CSS
- Responsive design

## Objetivo
Crear una web simple, elegante y funcional para emprendimientos que venden a través de redes sociales y WhatsApp.
